# 🛍️ RetailPulse

## AI-Powered Retail Analytics & Demand Forecasting Platform

RetailPulse is an AI-powered retail analytics platform that helps businesses gain actionable insights from retail sales data. It combines machine learning, data analytics, and interactive dashboards to support business decisions such as sales forecasting, customer segmentation, inventory optimization, and customer churn prediction.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/<your-username>/RetailPulse.git
cd RetailPulse
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run streamlit_app/app.py
```

---

## 🐳 Docker

Build Docker Image

```bash
docker build -t retailpulse .
```

Run Container

```bash
docker run -p 8501:8501 retailpulse
```
